##
### How to install
```Bash
$ pkg install git
$ pkg install python
$ pip install requests
$ git clone https://github.com/KZREVOXTICAL/ProxyDownloader
$ cd ProxyDownloader
$ python main.py
```
### Proxy
- [x] http
- [ ] https
- [x] socks4
- [X] socks4

### Example of using download proxy
```Bash
python main.py -p socks4 -t 200
```

### Example of using download proxy
```Bash
python main.py -f listproxy -o live.txt
```
